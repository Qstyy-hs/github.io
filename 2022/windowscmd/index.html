计算机基础——Windows命令

2022-09-26
Windows
  • Windows
N 人看过
    • 网络相关指令

  • ipconfig

  • 1.查看本机IP：ipconfig

  • 2.查看IP地址等网卡配置：ipcofig /all

  • 3.清除本地DNS缓存：ipconfig /flushdns

  • Ping

  • 1.检测主机是否能通信：ping 目标ip地址

  • 2.查看访问该ip/域名的延迟和丢包率：ping ip/域名

  • 3.ping ip/域名m次：-n m

  • 4.一直ping一个ip/域名：-t

  • 5.路由追踪：tracert ip/域名

  • 路由追踪Tracert主要是用来确定ip报文访问目标的时候所经过的路径

  • 6.测试远程主机端口能否正常通信：talnet ip/域名 端口

  • 使用前先在Windows功能里面开启”Telnet客户端”或者”Telnet Client”这个功能 例如：telnet 192.168.0.55 8080

  • 7.设置防火墙规则：wf.msc

  • Whoami

  • 查看当前用户及权限：whomai

  • Systeminfo

  • 查看计算机信息（版本、位数、补丁情况）

  • Ver

  • 查看计算机操作系统版本

    • 用户和组

  • Net

  • 1.查看当前系统有哪些用户：net user

  • 2.查看用户的基本信息，所属组：net user 用户名

  • 3.添加新用户并设置密码：net user 用户名 密码 /add

  • 4.删除用户：net user 用户名 /del

  • 5.查看当前在线的用户：query user

  • 6.查看所有的本地组：net localgroup

  • 7.查看administrators组中用户：net localgroup administrators

  • 8.将用户xxx添加到本地管理员：net localgroup administrators xxx /add

  • 9.判断当前用户是否是域成员：net user /domain

  • Wmic

  • WMIC扩展WMI（Windows Management Instrumentation，Windows管理工具） ，提供了从命令行接口和批命令脚本执行系统管理的支持。在cmd中有些时候查到的数据不全，如某些进程的pid，这时可以使用wmic进行操作，WMIC提供了大量的全局开关、别名、动词、命令和丰富的命令行帮助增强用户接口。

  • 查看启动程序信息：wmic startup get command,caption

  • 查询本机服务信息：wmic service list brief

  • 查询进程信息：tasklist

  • 查看计划任务：schtasks /query /fo LIST /V

  • 查看端口列表：netstat -ano

  • 注意，一般查看进程端口，先查进程pid，在根据pid查端口

  • 文件和文件夹

      • 文件夹

  • 1.进入D盘：d:

  • 2.切换磁盘和目录：cd /d cd c:/xxx

  • 3.进入文件夹：cd \test1\test2

  • 4.返回根目录：cd \

  • 5.返回上级目录：cd ..

  • 6.新建文件夹：md 文件夹名

  • 7.显示目录中文件列表：dir

  • 8.显示目录结构：tree c：\test

  • 9.显示当前目录位置：cd

  • 10.切换指定磁盘的当前目录位置：cd d:

  • 11.删除文件：rd 文件夹

  • 12.复制文件：copy 路径\文件名 路径\文件名

  • 13.移动文件：move 路径\文件名 路径\文件名

  • 14.删除文件：del 文件名
